# Regulatory Positioning

This repository is a research-software project for ultrasound signal-processing
methods. It is not clinical decision support software.

Key constraints:

- No output from this repository should be interpreted medically.
- The codebase does not claim clinical benefit, diagnostic improvement,
  regulatory clearance, or patient outcome improvement.
- RQM Technologies is not positioned here as an ultrasound hardware
  manufacturer; this work is software-method research.
- Any future clinical use would require independent validation, quality
  management controls, regulatory review, and appropriate
  clearance/authorization.
