# Validation Ladder (Research to Potential Translation)

Validation is staged to keep research claims bounded:

1. Synthetic unit tests
2. Known-geometry point targets
3. Conventional delay-and-sum baseline parity
4. Public/open research ultrasound datasets where licensing allows
5. OEM/academic expert review
6. Regulatory pathway separation if ever productized

All current outputs are for software-method evaluation and benchmark reporting.
