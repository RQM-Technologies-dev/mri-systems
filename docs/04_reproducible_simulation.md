# 04 - Reproducible simulation

Simulations in this repository are controlled synthetic demonstrations, not
clinical performance reports.

Each simulation folder includes:

- `run.py`
- `config.yaml`
- output directory creation
- deterministic synthetic generation
- baseline and coherence-aware placeholder outputs
- `metrics.json`
- `summary.md`

Expected artifacts per run:

- baseline reconstruction
- quaternionic/coherence-aware output
- error map
- coherence defect map
- metrics report

Simulation outputs should be treated as evidence-building scaffolds for offline
evaluation workflows.
