# Phase 1 Benchmark Plan

Initial benchmark tracks for this research prototype:

1. Single point target localization
2. Multiple scatterer reconstruction consistency
3. Noise sensitivity of RF simulation and beamforming outputs
4. Channel dropout robustness for baseline beamforming
5. Conventional coherence factor vs quaternionic alignment metrics
6. Runtime scaling by `n_elements` and beamforming grid size

These benchmarks are method-comparison artifacts only and are not evidence of
clinical efficacy or regulatory readiness.
